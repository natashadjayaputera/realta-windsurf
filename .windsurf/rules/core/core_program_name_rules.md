# Program Name Format
`AABXXXYY`

Exactly 5 characters long.

Where:
- AA: Module Name
- B: Type of program
- XXX: Program Number
- YY: Sub Program Number

List of type of program:
- B: Batch
- R: Report
- M: Master
- T: Transaction
- I: Inquiry

For Example:
`FAM00500`
Means:
- Module Name: FA
- Type of program: M - Master
- Program Number: 005
- Sub Program Number: 00
