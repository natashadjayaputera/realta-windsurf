# Back Function Categories
1. CategoryName: `business-object-overridden-function`
- Has `protected override` keyword
- Function name start with `R_` 
2. CategoryName: `batch-function`
- Function name is `R_BatchProcess` or `R_BatchProcessAsync`
3. CategoryName: `other-function`
- Any function that is not business object overridden function or batch function
