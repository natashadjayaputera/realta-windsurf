# Solution Structure Enforcement

- **MUST** maintain hierarchical folder structure in solution  
- **MUST** nest projects under correct module folders  
- **NEVER** place projects in root solution folder  
- **NEVER** mix modules in the same folder  
- **NEVER** mix backend and frontend in one solution
